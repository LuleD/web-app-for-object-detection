Machine Learning Concept : Object Detection

Multiple Object Detection

This Proof of Concept is based on a Machine Learning Model trained using Tensorflow. The Model can identify different objects with a high degree of accuracy ~ >80%.

To test it, once this page loads wait for 1-2 minutes for the Model to load. Once its ready, the "Enable Webcam" button below will be clickable. Click the button to test the functionality of the model.

Hold some objects up close to your webcam to get a real-time classification! When ready, click "Enable Webcam" below and grant access to the webcam when the browser asks (check the top left of your window).
